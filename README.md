# Fract'ol
Fractal Renderer - 42 Graphic branch project 

<img src="gitimg/julia.gif" width=600 >

# Built & Use

+ C 
+ MLX Library
+ Runs on OS X and Linux.
+ Multithread

```
make && ./fractol [julia, mandelbrot, tribrot, burningship, lyapunov]
```
